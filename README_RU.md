# Poizon Price Calculator Bot

Телеграм-бот для автоматизации расчета стоимости товаров с площадки Poizon. Проект разработан как готовое решение для байеров и логистических компаний.

## Основной стек технологий
- Python 3.12
- aiogram 3.x (Асинхронная разработка)
- SQLAlchemy 2.0 (ORM для работы с БД)
- PostgreSQL (Реляционная база данных)
- Docker & Docker Compose (Контейнеризация)

## Ключевой функционал
- Машина состояний (FSM): интуитивный процесс выбора категории и ввода цены.
- Динамический расчет: стоимость формируется по формуле с учетом курса юаня, комиссии и логистики.
- Логирование в БД: каждый расчет сохраняется в PostgreSQL для последующей аналитики.
- Админ-панель: мгновенные уведомления администратора о новых запросах.

## Формула расчета
Итоговая цена (RUB) = (Цена в юанях * Курс) + Доставка + Комиссия
$$Total = (Price_{CNY} \times Rate) + Shipping + Commission$$
